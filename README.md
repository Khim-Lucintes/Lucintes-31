## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Customization

### Update Personal Information

1. **Hero Section**: Edit `src/components/Hero.tsx` to change your name and description
2. **About Section**: Update `src/components/About.tsx` with your information
3. **Skills**: Modify the skills array in `src/components/Skills.tsx`
4. **Projects**: Update the projects array in `src/components/Projects.tsx`
5. **Contact**: Change social links and contact info in `src/components/Contact.tsx`

### Styling

All styles use CSS variables defined in `src/index.css`. You can customize:
- Colors (primary, secondary, background, etc.)
- Spacing
- Border radius
- Transitions

### Theme Colors

Edit the CSS variables in `src/index.css`:

```css
:root {
  --primary-color: #6366f1;
  --secondary-color: #ec4899;
  --background: #0f172a;
  /* ... more variables */
}
```
